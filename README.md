# 3D Portfolio Website

A luxurious, 3D-enabled portfolio website built with Next.js, Tailwind CSS, and React Three Fiber.

## Features

- **3D Hero Scene**: Interactive 3D elements.
- **Smooth Animations**: Scroll-triggered animations with Framer Motion.
- **Modern UI**: Glassmorphism, custom cursor, and responsive design.

## Getting Started

1. **Install dependencies**:
   ```bash
   pnpm install
   ```

2. **Run the development server**:
   ```bash
   pnpm dev
   ```

3. **Open [http://localhost:3000](http://localhost:3000)** with your browser to see the result.

## Customization

- **Content**: Update sections in `src/components/sections`.
- **Styles**: Modify `src/app/globals.css`.
- **3D Models**: Edit `src/components/3d`.

## Deploy

Deploy easily on [Vercel](https://vercel.com/new).
